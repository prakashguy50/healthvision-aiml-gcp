# ADR 003: Vertex AI for Model Serving

## Status
Accepted

## Context
Need to serve:
- Multiple model versions
- High-throughput predictions
- Auto-scaling

## Decision
Use Vertex AI because:
- Managed infrastructure
- Built-in monitoring
- Version control

## Consequences
- ✅ Reduced operational overhead
- ✅ Automatic scaling
- ❄️ Vendor lock-in
